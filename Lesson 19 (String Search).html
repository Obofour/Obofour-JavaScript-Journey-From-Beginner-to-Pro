JavaScript String indexOf()

The indexOf() method returns the index(position) of the first occurence of a string in a string or it returns -1 if the string is not found

JavaScript string lastindexOf()

The lastindexOf() method returns the index of the last occurence of a specified text in a string

The lastIndexOf() method searches backwards (from the end to the beginning),meaning:if the second parameter is 15,the search start at position 15 and searches to the beginningof the string

JavaScript string search()

The search() method searches a string for a string (or a regular expression) and return the position of the match

Did You Notice

The two method, indexOf() and search(),are equal?

They accept the same argument(parametrs) and return the same value

    The two method are not equal.These are the differences:

  • The search() method cannot take a second start position argument
  • The indexOf() method cannot take powerful search values (regular expression)

    • You will learn more about regular expression in a later chapter
